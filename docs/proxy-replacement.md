# Proxy Replacement

You can replace and list the replaced proxies using the Proxy Replacement API. There are 2 main endpoints for replacing proxies.

## Proxy replacement

Set of rules indicating which proxies to replace from and replace with in a proxy list.
[See Proxy Replacement Object](/proxy-replacement/proxy_replacement).

### Replaced proxy

List of replaced proxies which can be filtered by proxy replacement or the proxy list.
[See Replaced Proxy Object](/proxy-replacement/replaced_proxy).
